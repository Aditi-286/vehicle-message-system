# vehicle-message-system

Vehicle Message System is a Python GUI application developed using the Kivy framework. This project simulates a smart vehicle dashboard messaging interface where users can view different types of vehicle-related alerts and notifications in an organized and interactive way.

The application uses Kivy’s ScreenManager feature to manage multiple screens efficiently. It contains a message list screen that displays various vehicle notifications and a detail screen that shows complete information about the selected message. Users can easily navigate between screens using buttons and interactive controls.

The project demonstrates important GUI development concepts such as:

* Screen navigation using ScreenManager
* Dynamic button creation
* Event handling with button bindings
* Layout management using BoxLayout
* Interactive user interface design
* Data handling using Python dictionaries and lists

The application includes several sample vehicle notifications such as:

* Vehicle maintenance reminders
* Fuel level alerts
* Battery health warnings
* Navigation route updates
* Welcome and system messages

This project is designed for beginners learning Python GUI development and Kivy application design. It helps in understanding how real-world dashboard systems manage and display notifications inside modern vehicles.

## Features

* Interactive graphical user interface
* Multiple screens with smooth navigation
* Dynamic message display system
* Clean and simple dashboard layout
* Easy-to-understand Python code structure
* Beginner-friendly Kivy project

## Technologies Used

The main purpose of this project is to practice GUI application development using Kivy and understand how screen management and event-driven programming work in Python applications.
